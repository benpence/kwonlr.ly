Javascript Review

Setting a variable equal to a value: 

> var five = 5
> five
5

> var oneAndFiveList = [1, five]
> oneAndFiveList
[1, 5]
> oneAndFiveList.length
2
> oneAndFiveList[0]
1
> oneAndFiveList[1]
5

> var bob = { name: "Bob", "age": 33 };
> bob.name
"Bob"
> bob["name"]
"Bob"
> bob.age
33

The general structure of a function: 

function isEqualToFive (theNumber) {
  return theNumber == 5;
}

function isEqualToFive (aNumber) {
  return 5 == aNumber;
}

Note that the two function definitions above are the same.

Value Functions

Suppose you have a number var one = 1;

  1. Write a function called addOne which takes in a number and returns that number plus 1. 
    > addOne(one)
2
> addOne(2)
3
  2. Write a function called isGreaterThanThree which takes in a number and returns whether or not that number is greater than 3. 
    > isGreaterThanThree(one)
false
> isGreaterThanThree(3)
false
> isGreaterThanThree(4)
true

List Functions

Suppose you have a list var list = [1, 2, 3, 4, 5];

  1. Write a function called inList which takes in a number and returns whether or not that number is in list. 
    > inList(1)
true
> inList(2)
true
> inList(7)
false
  2. Write a function called addOneToAll which takes in a list of numbers and returns a new list with each number from the old list incremented by 1. 
    > addOneToAll(list)
[2, 3, 4, 5, 6]
> addOneToAll([0, 0, 0])
[1, 1, 1]
> addOneToAll([])
[]
  3. Write a function called onlyGreaterThanThree which takes in a list of numbers and returns a new list with each number from the old list that is greater than 3. 
    > onlyGreaterThanThree(list)
[4, 5]
> onlyGreaterThanThree([0, 0, 0])
[]
  4. Write a function called sum which takes in a list of numbers and returns the sum of all the numbers in the list. 
    > sum(list)
15
> sum( addOneToAll(list) )
20
> sum([0])
0
